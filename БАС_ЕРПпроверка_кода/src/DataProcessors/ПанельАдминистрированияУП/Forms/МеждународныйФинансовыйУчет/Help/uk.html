Международный финансовый учет

Используя данный раздел панели администрирования можно включить функциональные возможности подсистем бюджетирование и международный финансовый учет.

Бюджетирование. Подсистема предназначена для многовариантного (многосценарного) планирования денежных средств и финансовых показателей хозяйственной деятельности, как правило, на срок не менее месяца.

Основная цель подсистемы заключается в организации процесса управления через бюджетные формы, данные которых, позволяют получить четкую картину как текущей, так и будущей финансовой ситуации на предприятии с точки зрения оценки выбранных показателей (прибыли, остатков денежных средств и т.д.).

Использовать оповещения для задач бюджетирования - включает оповещения для задач бюджетирования, будут формироваться следующие оповещения:

  • Поступление задачи
  • Подходит срок выполнения задачи
  • Задача просрочена

Для включения в письмо адреса задачи можно заполнить поле Адрес публикации информационной базы в локальной сети

Флаг Автоматически запускать фоновое задание расчета хранимых данных при проведении документов - если среди правил получения фактических данных по статьям бюджетов есть правила отмеченные флагом Хранить результат выполнения правила, то включение этого флага позволяет актуализировать хранимые данные бюджетирования сразу после проведения документа. После проведения документа будет запущено фоновое задание, которое отразит движения документа в бюджетировании.

Флаг Автоматически запускать фоновое задание расчета хранимых данных из отчетов - если среди правил получения фактических данных по статьям бюджетов есть правила отмеченные флагом Хранить результат выполнения правила, то включение этого флага позволяет актуализировать хранимые данные бюджетирования при формировании отчетов. После формирования отчета будет запущено фоновое задание, которое отразит все неотраженные документы в бюджетировании. После завершения фонового задания пользователю будет предложено переформировать отчет.

Международный финансовый учет. Подсистема предоставляет возможность решать следующие задачи:

  • Отражение документов оперативного учета в международном финансовом учете;
  • Подготовка отчетности в соответствии с принципами международных стандартов финансовой отчетности (МСФО), включая возможность настройки дополнительных отчетов для проверки и анализа учетных данных;
  • Подготовка отчетности по правилам управленческого учета, построенным на основе МСФО;
  • Гибкая настройка форм финансовой отчетности.


Для работы с подсистемой в отношении международной финансовой отчетности необходимо выбрать из списка Функциональную валюту и Валюту представления.

В соответствии с п.8 МСФО (IAS) 21 «Влияние изменений обменных курсов валют» функциональная валютой является валюта, используемая в основной экономической среде, в которой предприятие осуществляет свою деятельность.

Валюта представления отчетности - это валюта, в которой финансовая отчетность представляется пользователям.

Возможность использования документов параллельного учета основных средств и нематериальных активов регулируется функциональной опцией Учет внеоборотных активов.
