Помощник выгрузки и загрузки модели бюджетирования

Помощник выгрузки/загрузки модели бюджетирования предназначен для обеспечения обмена данными бюджетной модели между несколькими информационными базами.

Инструменты реализации задачи обмена:

  • Выгрузка модели бюджетирования в файл для обмена с другими информационными базами;
  • Загрузка модели бюджетирования из файла, содержащего данные загружаемой модели бюдежетирования;
  • Копирование модели бюджетирования в рамках одной информационной базы.
