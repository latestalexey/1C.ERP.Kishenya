Ошибки настройки шаблонов проводок (международный учет)

Отчет предназначен для анализа и завершения настроек, сделанных в Шаблонах проводок.

Особенности отчета:

  • Отчет информирует об ошибке, если при каком-либо счете, выбранном по умолчанию или уточненном через Группу финансового учета, созданы шаблоны, отражающие движение только по дебету, или только по кредиту данного счета.
  • Для сужения фокуса анализа в отчете возможно применение отборов по Группам финансового учета или Счетам по умолчанию, а также отбора по иерархии справочников.
