Плановая потребность в полуфабрикатах

Отчет предназначен для анализа плана производства по полуфабрикатам (необходимых для его исполнения).

Отчет формируется из документа план производства с отбором по периоду планирования, сценарию и номенклатуре.

Отчет отражает потребность в полуфабрикатах, необходимых для исполнения плана производства (в каком количестве и в каком периоде.)
Расшифровки отчета позволяют оценить потребность в разрезе продукции.
