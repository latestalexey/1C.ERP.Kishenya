Настройка передачи материалов в производство

Настройка определяет порядок обеспечения производства материалами.

Возможны два варианта передачи материалов

  • По заказу на производство – для материалов, обеспечение которыми происходит по заказу на производство.
  • По заказу на внутреннее потребление - для материалов, наличие которых в производстве подразумевается постоянно. Например, этот вариант можно использовать для дешевых расходных материалов, потребляемых в больших количествах (смазочные материалы, гвозди и т.п.).

Помимо варианта передачи материалов указывается склад-отправитель, с которого по умолчанию будет происходить получение материалов.

Настройка передачи материалов определяет значение флага Заказать на склад в заказах на производство и маршрутных листах.
В варианте По заказу на производство - флаг будет автоматически устанавливаться, склад будет автоматически заполняться указанным складом-отправителем.
