Нормативно-справочная информация

Необходимым и достаточным условием для осуществления производственных задач является подготовка нормативно-справочной информации в контексте определения Ресурсных спецификаций, которые описывают параметры производственного процесса изготовления продукции и полуфабрикатов, а также отражают перечень входящей и исходящей потребности в продукции, работах, трудозатратах (по видам работ), материалах, этапах производственного процесса.

Количество одновременно действующих для номенклатуры ресурсных спецификаций не ограничено, но только одна из них определяется в текущий момент времени как Основная и используется в дальнейшем при формировании документов по умолчанию.

При необходимости нормативно-справочные данные могут быть детализированы сведениями:

  • Маршрутные карты - описывают обособленные группы технологических операций с указанием параметров их исполнения. В случае пооперационного планирования маршрутная карта является обязательным атрибутом этапа ресурсной спецификации, детализируя ресурсы до технологических операций.
  • Бригады - перечень исполнителей совместно выполняющих задание на производство. Включение работников в состав бригады выполняется без ограничения в выборе подразделения, в которое участник бригады принят на работу, без учета системы оплаты труда, установленной для работника.
  • Виды работ сотрудников - наименование видов работ востребованных в производственном процессе. Перечень видов работ может содержать кроме описания самих работ, еще и требуемый разряд рабочих и их профессию.
  • Структура рабочих центров - создание моделей рабочих центров для планирования производственных работ. Под рабочими центрами подразумеваются, например. единицы оборудования, выделенные рабочие места и т.д.
  • Разрешение на замену материалов - формирование перечня аналогов для используемых в производстве материалов.
  • Параметры межоперационных переходов - формирование параметров перенастроек оборудования для выполнения производственных задач.
