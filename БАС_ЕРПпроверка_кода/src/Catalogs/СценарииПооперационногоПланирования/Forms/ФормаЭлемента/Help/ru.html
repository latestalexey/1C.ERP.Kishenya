Сценарий пооперационного планирования

Справочник позволяет описать сценарии изменения параметров производственных подразделений и выполнить расчет пооперационного расписания с учетом данных настроек. При помощи справочника можно смоделировать следующие ситуации: увеличение парка оборудования, уменьшение парка оборудования, а так же изменение графика работы рабочих центров.
Для использования сценария при пооперационном планировании расписания, сценарий необходимо указать в параметрах расчета расписания.

Закладка Состав рабочих центров

Закладка Графики работы

Закладка Состав рабочих центров


На закладке определяются параметры изменения состава оборудования.

В табличной части Увеличение парка оборудования можно описать дополнительные рабочие центры, которые будут загружены в процессе расчета расписания производства. Количество дополнительных единиц оборудования указывается в колонке Количество. В колонке Аналог выбирается рабочий центр, определяющий параметры дополнительных рабочих центров.

В табличной части Уменьшение парка оборудования определяется перечень рабочих центров, которые не будут загружаться при пооперационном планировании расписания производства.

Закладка Графики работы


На закладке осуществляется настройка графиков работы оборудования. При заполнении табличной части действуют следующие правила:

  • Если заполнен рабочий центр, то выбранный график работы будет использован для данного рабочего центра;
  • Если заполнен вид рабочего центра и не заполнен рабочий центр, то выбранный график будет использован для всех рабочих центров данного вида;
  • Если не заполнен вид рабочего центра и рабочий центр, то выбранный график будет использован для всех рабочих центров данного подразделения.

Пример
Имеется подразделение с высоким уровнем загрузки оборудования, работающее по односменному графику. В качестве методики управления маршрутными листами для подразделения выбран способ Пооперационное планирование. Требуется оценить целесообразность перевода подразделения на двухсменный график работы. Для решения этой задачи можно создать новый сценарий и в таблице на вкладке Графики работы определить для данного подразделения двухсменный график работы. Затем следует выполнить расчет пооперационного расписания данного подразделения, выбрав в параметрах расчета созданный сценарий. Расписание будет построено с учетом двухсменного графика работы, указанного в сценарии.
