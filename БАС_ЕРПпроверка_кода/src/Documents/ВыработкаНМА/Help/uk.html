Выработка НМА

Документ Выработка НМА предназначен для регистрации объема выработанной продукции (выполненных работ) для начисления амортизации нематериальных активов.

В документе уточняется объем выработанной продукции (выполненных работ) по каждому нематериальному активу.

  • Нематериальный актив - нематериальный актив.
  • Объем работ - объем выработанной продукции (выполненных работ).
