Переоценка ОС

Документ предназначен для отражения в системе фактов переоценки основных средств.

При вводе документа уточняются реквизиты:

  • Организация -организация, в которой приняты к учету основные средства.
  • Местонахождение ОС - подразделение организации, в котором приняты к учету основные средства.
  • Событие ОС - наименование события переоценки основного средства, которое отражается документом.
  • Статья расходов, Аналитика расходов - направление начисления расходов в случае уценки
  • Статья доходов, Аналитика доходов - направление начисления доходов в случае дооценки после уценки


В табличной части уточняется список основных средств, предназначенных для переоценки. Для каждого ОС вручеую указывается Справедливая стоимость.

Для быстрого заполнения документа группой однотипных основных средств нужно ввести в табличное поле хотя бы один такой объект и автоматически заполнить табличное поле по кнопке Заполнить - По наименованию.
