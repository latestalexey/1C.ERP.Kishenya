Распределение расходов на себестоимость продукции

Документ предназначен для настройки распределения постатейных производственных расходов на себестоимость продукции, а также отнесения затрат, накопленных по статье расходов, на статьи с другим вариантом распределения.

Доступен следующий список вариантов распределения:

  • По подразделениям и этапам по правилам - Позволяет выбрать и настроить отдельно правила распределения по подразделениям и этапам.
  • По подразделениям вручную, по этапам по правилам - Позволяет указать вручную подразделения и доли стоимости, которые нужно отнести на них, а также выбрать и настроить правило распределения по этапам внутри указанных подразделений.
  • По этапам по правилу (в данном подразделении) - Выбор и настройка правила распределения по этапам внутри подразделения, где возник распределяемый расход.
  • По этапам по правилу (по всем подразделениям) - Позволяет выбрать и настроить правило распределения по этапам всех подразделений, где происходил производственный процесс.
  • По этапам вручную по всем подразделениям - Ручное распределение расхода с точностью до заказа и кода строки, или выпуска без распоряжения и кода строки.
  • На другие статьи расходов - Возможность отнести распределемый расход на статьи расходов с другим вариантом распределения, например в целях переквалификации расхода.

В зависимости от выбранного варианта распределения расходов в поле шапки документа Расходы распределяются меняется состав закладок документа, позволяя указывать и настраивать правила распределения по подразделениям, по этапам производства и выпускам без распоряжений, списания на другие статьи расходов, распределения вручную.

Основное

Закладка содержит основные реквизиты документа Номер, Дата, Комментарий и пр.

По подразделениям по правилу

Позволяет указать долю стоимости, распределяемую между подразделениями по указанному правилу. При выборе правила его настройки копируются в документ и могут быть изменены пользователем. Также есть возможность по нажатию на соответствующую кнопку на данной закладке сохранить измененные настройки в качестве настроек по умолчанию для выбранного правила распределения расходов по подразделениям.

По подразделениям вручную

Содержит таблицу, в которой можно вручную указать, сколько долей стоимости нужно отнести на определенное подразделение, по указанной в таблице статье калькуляции.

По этапам по правилу

На данной закладке можно указать правило, по которому будут распределяться расходы по этапам заказов и выпускам без распоряжений. Аналогично правилу распределения по подразделениям, настройки выбранного правила копируются в документ и могут быть изменены пользователем. Также становится возможным указание списка групп (видов) продукции, который будет использован как фильтр для базы распределения статьи расходов. (Данная настройка возможна, если включена ФО Администрирование/Производство и ремонты/Аналитический учет по группам (видам) продукции).
Если расход не распределяется вручную по подразделениям, то на данной закладке доступен выбор статьи калькуляции. Значение данного элемента управления автоматически заполняется из одноименного реквизита правила распределения по этапам, а при его не заполненности в правиле, из данных статьи расходов.
Гиперссылка Проверить наличие данных для базы распределения открывает контекстный отчет, который показывает наличие данных, на основании которых будет формироваться база распределения.

По этапам заказов вручную

Содержит таблицу, в которой можно указать этапы заказов, на которые можно отнести расходы в указанной доле стоимости.

По выпускам без распоряжений вручную

Содержит таблицу, в которой указываются документ выпуска без распоряжений и код строки, на которые необходимо отнести распределяемые расходы в указанной доле стоимости

Дополнительно на другие статьи расходов

Возможность отнесения затрат накопленных по распределяемой статье на статьи с другим вариантом распределения. Данная возможность может быть использована для переквалификации расходов.

Распределение на другие статьи расходов доступно при любом варианте распределения, позволяя отнести часть расхода на другие статьи.
Указание доли стоимости необходимо для предварительного распределения расходов до момента расчета фактической себестоимости расхода. Фактическая себестоимость расхода рассчитывается после выполнения всех операций по закрытию месяца. После расчета фактической себестоимости данные по распределению расходов будут уточнены.

В заголовках закладок документа отображается информация о сумме долей стоимости, распределенных в соответствии с настройками на этих закладках.
