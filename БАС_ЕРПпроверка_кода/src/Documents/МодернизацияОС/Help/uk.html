Модернизация ОС

Документ предназначен для отражения модернизации (реконструкции) основных средств.

В документе заполняются следующие реквизиты:

  • Местонахождение ОС - подразделение организации, в котором приняты к учету основные средства
  • Событие - событие основного средства, которое отражается документом.


На закладке Основные средства указывается список модернизируемых основных средств.

Для быстрого заполнения документа группой однотипных основных средств нужно ввести хотя бы один такой объект и автоматически заполнить табличное поле основными средствами с таким же наименованием по кнопке Заполнить - По наименованию.

Для изменения параметров учета ОС, изменившихся после выполнения модернизации, используется документ Изменение параметров ОС.
В документе модернизации для удобства работы предусмотрен его ввод на основании.

Сумма модернизации не указывается в документе. При проведении модернизации стоимость каждого объекта эксплуатации увеличивается на сумму, накопленную для этого объекта по счетам 151, 1522, 1532.
