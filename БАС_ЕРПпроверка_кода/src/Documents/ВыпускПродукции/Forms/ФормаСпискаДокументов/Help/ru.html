Оформление выпуска продукции по заказу

На закладке К оформлению выводится список распоряжений, по которым требуется оформить выпуск продукции.
Список распоряжений заполняется автоматически на основании других документов. Например, выполненные маршрутные становятся распоряжениями на выпуск продукции.

Для оформления выпуска по заказу предназначена команда Оформить выпуск.

Для оформления одного выпуска продукции сразу по нескольким распоряжениям необходимо предварительно включить функциональную опцию Выпуск продукции, работ и услуг по нескольким распоряжениям в разделе НСИ и администрирование - Производство.

Оформление выпуска продукции без заказа

Для оформления выпуска предназначена команда Создать выпуск без заказа.
