Сверка данных оперативного и бухгалтерского учета

Отчет предназначен для выявления и представления в табличном формате отклонений остатков и/или оборотов оперативного учета от регламентированного бухгалтерского учета за отчетный период.
Отчет может быть использован как инструмент аудита для решения следующих задач:

  • Просмотр оборотов оперативного учета и проведение сравнительного анализа отклонений в суммах оборотов оперативного и регламентированного бухгалтерского учета с разбивкой по Разделам учета, обосновывающих разницу в регламентированном и оперативном учете.
  • Расшифровка показателей с детализацией до документа, которая показывает какой документ вызвал расхождения.

Разделы учета

Отчет структурирован по следующим разделам учета:

Денежные средства

Себестоимость товаров

Расчеты с поставщиками (Расчеты с клиентами)

Расчеты по кредитам и депозитам

Прочие расчеты с контрагентами

Расходы

Денежные средства


Источники данных

  • Регистр накопления «Денежные средства (безналичные)»
  • Регистр накопления «Денежные средства (наличные)»
  • Регистр накопления «Денежные средства в кассах ККМ»
  • Регистр накопления «Денежные средства у подотчетных лиц»
  • Регистр накопления «Денежные средства в пути»
  • Регистр бухгалтерии «Хозрасчетный» (счета 30, 31, 33, 372)


Особенности

  • Сверка производится только в валюте регламентированного учета
  • Субконто "Касса" по счету 30 нет, поэтому сверка проводится по всем кассам предприятия

Себестоимость товаров


Источники данных

  • Регистр накопления «Себестоимость товаров» (измерение «Раздел учета» не равно «Производственные затраты», «Товары принятые на комиссию», «Материалы принятые в переработку»)
  • Регистр бухгалтерии «Хозрасчетный» (все балансовые счета, у которых субконто «Номенклатура» и «Склад»)


Особенности

  • Методика отражения движений в регистрах оперативного учета и в регламентированном отличается, поэтому для документов, отражающих хозяйственные операции «ВозвратИзПроизводства», «ВозвратТоваровОтКлиента», «ВозвратОтКомиссионера», «ВозвратОтПереработчика» для оперативного учета «Приход» и «Расход» поменяны местами с изменением знака.
  • Субконто «Склад» на субсчетах учета номенклатуры оборотное. Для корректной работы отчета нужно исправить на необоротное.

Расчеты с поставщиками (Расчеты с клиентами)


Источники данных

  • Регистр накопления «Расчеты с поставщиками по документам» («Расчеты с клиентами по документам»)
  • Регистр бухгалтерии «Хозрасчетный» (счет 63 (371) с субсчетами)


Особенности

  • Сравнение ведется только в валюте регламентированного учета
  • Методика отражения возвратов в оперативном и регламентированном учетах разная – в регламентированном учете «сторно», в оперативном «расход». Поэтому обороты по оперативному учету откорректированы таким образом, что бы они совпадали с бухгалтерским.
  • В оперативном учете отсутствует необходимость в движениях, закрывающих авансы. Соответственно в обороты оперативного учета при необходимости добавляется «виртуальный» оборот, имитирующий операцию закрытия аванса в регламентированном учете.

Расчеты по кредитам и депозитам


Источники данных

  • Регистр накопления «Расчеты по договорам кредитов и депозитов»
  • Регистр бухгалтерии «Хозрасчетный» (счета: 313, 314, 377, 50, 60, 61) с отборам по договорам кредитов и депозитов


Особенности

  • Сверка производится только в валюте регламентированного учета

Прочие расчеты с контрагентами


Источники данных

  • Регистр накопления «Расчеты с поставщиками по документам»
  • Регистр накопления «Расчеты с клиентами по документам»
  • Регистр бухгалтерии «Хозрасчетный» (счета 377, 685 с отборам по договорам купли-продажи)


Особенности

  • Сравнение ведется только в валюте регламентированного учета

Расходы


Источники данных

  • Регистр накопления «Себестоимость товаров» (когда измерение «Раздел учета» = «Производственные затраты»)
  • Регистр накопления «Себестоимость товаров» (с видом номенклатуры «Работа»)
  • Регистр накопления «Прочие расходы»
  • Регистр накопления «Прочие расходы незавершенного производства»
  • Регистр накопления «Трудозатраты незавершенного производства»
  • Регистр накопления «Трудозатраты на выпуск без распоряжений»
  • Регистр бухгалтерии «Хозрасчетный» (счета 20 ДР, 23, 24, 28 ДР, 91-97)


Особенности

  • Аналитика на счетах затрат – оборотная (кроме вида субконто «Группы (виды) продукции»), поэтому сверка производится без аналитики (но с расшифровкой до регистратора)
  • Статьи расходов с неиспользуемым более вариантом распеределения На прочие активы и аналитикой Прочие активы не анализируются по оперативному учету, как соответсвующие статьям активов. Статьи расходов с вариантом распределения На внеоборотные активы не имеют отражения по регламентированным суммам в оперативном учете и с бухгалтерским учетом так же не сравниваются.
