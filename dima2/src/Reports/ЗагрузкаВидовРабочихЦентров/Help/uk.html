Загрузка видов рабочих центров по графику производства

Предназначен для выбора оптимальной загрузки рабочих центров. Способствует выявлению потенциала по наращиванию объемов производства.

В отчете выводится следующая информация:

  • Свободно - свободное в интервале планирования время вида рабочих центров, доступное для размещения следующих этапов
  • Занято - время вида рабочих центров, занятое запланированными этапами графика производства

Данные о загрузке берутся из графика производства.

Занятость видов рабочих центров продукцией по графику производства

Предназначен для детализации занятости видов рабочих центров до производственных процессов выпуска конкретной продукции согласно графика производства.
