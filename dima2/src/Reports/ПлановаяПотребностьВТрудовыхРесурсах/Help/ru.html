Плановая потребность в трудовых ресурсах

Отчет предназначен для анализа плана производства по трудозатратам (необходимым для его исполнения).
Отчет отражает потребность в трудозатратах, необходимых для исполнения плана производства (в каком количестве и в каком периоде).
Расшифровки отчета позволяют оценить потребность в разрезе продукции.

Отчет формируется из документа план производства с отбором по периоду планирования, сценарию и номенклатуре.

Анализ трудозатрат с любыми единицами измерения выполняется в человеко-часах.
Для видов работ сотрудников с произвольным единицами измерения необходимо указывать данные трудоемкости в человеко-часах.
