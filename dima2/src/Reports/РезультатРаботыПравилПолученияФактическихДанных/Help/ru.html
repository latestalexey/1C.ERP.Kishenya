Результат работы правил получения фактических данных

Отчет предназначен для контроля результатов работы правил получения фактических данных, а также оценки фактических данных по статьям и показателям полученым при построении бюджетных отчетов.

Отчет вызывается из формы справочников Статьи бюджетов, Показатели бюджетов и из рабочего места Настройка правил получения фактических данных бюджетирования.
