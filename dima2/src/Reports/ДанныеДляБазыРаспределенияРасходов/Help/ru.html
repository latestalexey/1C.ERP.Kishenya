Данные для базы распределения расходов

Отчет предназначен для информирования о наличии данных для построения базы распределения расходов.

Особенности отчета:

  • Наличие данных для базы не означает, что база распределения гарантирована будет построена, т.к. при построении базы автоматически программно накладывается ряд фильров на данные, и т.д.
  • В некоторых случаях отчет может не показывать данные для построения базы, но это не означает, что расходы в итоге не распределятся. Например, если используется база распределения Сумма расходов на оплату труда, то данные для этой базы появятся в процессе выполнения операций партионного учета по отнесению трудозатрат на выпуски продукции.
