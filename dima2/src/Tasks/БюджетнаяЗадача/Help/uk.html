Бюджетная задача

Задача бюджетирования представляет собой задание на выполнение действий по бюджетному процессу. Задача формируется как часть Этапа бюджетного процесса, содержит ожидаемый срок и отметку о фактическом времени ее исполнения. В задаче находится ссылка на документ или отчет, которые заполняются в соответствии с настройкой действия из шага бюджетного процесса, например, документ Экземпляр бюджета по указанному Виду бюджета и Сценарию для ввода плановых данных.

По мере выполнения одних задач формируются новые задачи по следующим этапам процесса. Нарушение нормативных сроков исполнения текущих задач может привести к сдвигу всех дальнейших задач.

Для контроля над выполнением текущих задач и оценки времени, необходимого для исполнения последующих с учетом переноса сроков задач, в системе предусмотрено рабочее место Монитор бюджетных процессов.

По имеющимся задачам можно формировать связанные задачи на других исполнителей. Например, в рамках задания на проверку бюджета можно создать поручение на корректировку плана.

Сводная информация о выполнении задач ответственными лицами по этапам процесса отображается в отчете Статистика выполнения задач бюджетного процесса. На основании данного отчета можно сделать вывод об эффективности прохождения бюджетного процесса и разумности его нормативных сроков

Связанные задачи

Отчет показывает состояние выполнения задач, созданных на основании текущей.


Возможна рассылка оповещений о поступлении бюджетных задач и просрочке выполнения. Включение использования оповещений доступно в пункте меню НСИ и администрирование - Бюджетирование и планирование - Бюджетирование.
