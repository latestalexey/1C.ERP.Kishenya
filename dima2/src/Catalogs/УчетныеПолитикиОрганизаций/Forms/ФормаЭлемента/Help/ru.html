Метод оценки стоимости товаров

Для выбора доступны следующие методы оценки:

  • Средняя (взвешенная) за месяц;
  • ФИФО (взвешенная оценка);
  • ФИФО (скользящая оценка).

Метод ФИФО (скользящая оценка) доступен для выбора в случае, если включено использование партионного учета (раздел НСИ и администрирование - Финансовый результат и контроллинг - Учет товаров)

Система налогообложения

Выбирается одна из предопределенных Систем налогообложения, определяющая, плательщиком каких основных налогов является организация.

Определять объект налогообложения без корректировок финансового результата

При включенном флаге для организации-плательщика налога на прибыль при заполнении Декларации по прибыли начисляются разницы и заполняется Приложение РІ.
При выключенном флаге налогооблагаемая прибыль предприятия равна его финансовому результату.

Отражение расходов на оплату труда

Настройка определяет детализацию расчетов с работниками на счете 66 "Расчеты по выплатам работникам". Если флаг установлен, то документ Отражение зарплаты в финансовом учете должен содержать данные по работникам.

Включать сумму дооценок ОС пропорционально начислению амортизации в состав нераспределенной прибыли с уменьшением дополнительного капитала (в соответствии с п. 21 П(С)БУ 7 и п. 41 МСФО 16)

При включенном флаге сумма дооценок, накопленная на счете 41 "Капитал в дооценках", ежемесячно закрывается на нераспределенную прибыль
