Группы настроек финансового учета доходов/расходов

Справочник предназначен для классификации объектов по правилам отражения в бухгалтерском (финансовом) учете.

Варианты использования Группы настроек финансового учета доходов/расходов

  • Для объединения движений по статьям доходов и расходов, отражаемых на одинаковых счетах учета в международном учете.
  • Для получения счетов международного финансового учета при формировании проводок по документам движения по статьям доходов и расходов.
  • В шаблонах проводок при настройке уточнения счетов учета.
