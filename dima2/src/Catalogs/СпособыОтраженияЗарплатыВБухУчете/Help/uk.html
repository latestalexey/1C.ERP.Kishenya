Способы отражения зарплаты

Справочник Способы отражения зарплаты в финансовом учете предназначен для настройки статей расходов и активов / пассивов, по которым отражаются начисления сотрудникам в финансовом учете. В карточке способа задаются статьи и аналитики для начисленной зарплаты и взносов в фонды. Для расходов, относящихся к деятельности с особым порядком налогообложения, предусмотрен отдельный комплекс настроек.

Отнесение зарплаты на расходы

Большинство начислений принимается к учету как расходы предприятия. Для этого в способе отражения требуется указать статьи расходов для начислений и взносов. При отражении зарплаты счет регламентированного учета по дебету определяется подразделением и статьей расходов и настраивается в рабочем месте Отражение документов в регламентированном учете.

Отнесение на прочие активы / пассивы

В случаях, когда начисления выполняются за счет ранее сформированных резервов или по другим причинам не относятся на расходы предприятия, в способе отражения возможно указание статьи активов / пассивов. Для таких начислений счет регламентированного учета по дебету задается непосредственно в способе отражения. При отражении зарплаты в финансовом учете такое начисление будет отражено в учете прочих активов / пассивов и не будет отнесено на расходы предприятия.

Подбираемые аналитики

Аналитики следующих типов могут быть заполнены автоматически:

  • Организация
  • Подразделение
  • Объект эксплуатации
  • Работник

Подбор аналитики выполняется при отражении зарплаты в финансовом учете. Перечень подбираемых аналитик определяется данными документа отражения зарплаты в финансовом учете. Например, аналитика по работникам будет заполнена, если зарплата принимается к учету в разрезе сотрудников организации. Аналитика Объект эксплуатации заполняется по сдельным работам по заказам на ремонт, то есть по документам Выработка сотрудников.
