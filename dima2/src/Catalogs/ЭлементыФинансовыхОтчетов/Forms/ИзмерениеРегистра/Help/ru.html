Измерение регистра

Элемент отчета Измерение регистра позволяет вывести в бюджетный отчет все значения измерения из оборотов по статьям бюджетов и остатков по показателям бюджетов.

  • Представление - наименование элемента отчета для отображения в дереве элементов в виде бюджета
  • Имя измерения - имя измерения регистра бюджетирования, которое будет служить источником значений отчета
  • Комментарий - дополнительное описание, которое не будет отображаться в бюджетном отчете

Источником значений могут быть все статьи и показатели бюджетов, входящие в вид бюджета, либо только выбранные.
