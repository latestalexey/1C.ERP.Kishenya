Показатель бюджетов

Справочник предназначен для регистрации остаточных показателей (показателей бюджетов), формирующих прогнозный баланс.

Показатели бюджетов являются расчетными значениями суммы фактических остатков на начало периода и запланированных оборотов по статьям бюджета.

В момент создания связи между статьей бюджета и показателем бюджета определяется, каким образом статья влияет на показатель: увеличивает или уменьшает его целевое значение, и с каким коэффициентом.

Показатель бюджета характеризуется аналитическими измерениями, максимальное возможное количество измерений - 6. Показатель бюджета может быть не только стоимостным, но и количественным. Для этого предусмотрен флаг По клоичеству. При включении данного флага можно выбрать либо использование определенной единицы измерения, выбрав опцию Указанная единица измерения, либо определить единицу измерения из аналитики при выборе опции Единица измерения определяется аналитикой, аналогично с валютным учетом.

Закладка Настройка типа показателя определяет характер получения данных по показателям. Опция Расчетный позволяет производить автоматизированный расчет остатков, опция Целевой позволяет вводить данные вручную.

В отчет могут быть выведены не все остатки и обороты показателя, а только, значения, соответствующие установленным отборам.

  • Представление - наименование элемента отчета для отображения в дереве элементов в виде бюджета
  • Показатель - в отчет могут быть выведены Суммовое или Количественное значение показателя. В простой таблице можно указать Сумма и количество - тогда система отобразит оба показателя.
  • Гиперссылка Смещение и границы данных - открывает форму настройки периодов источника
  • Значение - позволяет выбрать, что нужно выводить в отчет: обороты и сальдо по показателю
  • Настройки получения данных - позволяют переопределить фильтры по Сценарию, Организации и Подразделению из бюджетного отчета. Используются, когда нужно вывести значение статьи бюджетов вне зависимости от настроек отчета. Например, для расчета % остатка по показателю по организации, выбранном в бюджетном отчете в оборотах по всей группе компаний
  • Дополнительный отбор - позволяет наложить дополнительный отбор на обороты по показателю по значениям аналитических разрезов или измерений
  • Комментарий - дополнительное описание, которое не будет отображаться в бюджетном отчете
