Группа

Элемент отчета Группа предназначен для:

  • Визуальной группировки статей бюджетов, показателей бюджетов и измерений бюджетного отчета
  • Отображения итогов с помощью элемента Формула по группе


Если элемент Формула по группе отображается на первом месте в группе, то итоговые значения будут отображены в строке, напротив элемента "Группа", иначе элемент Формула по группе будет отображен отдельной строкой.
