Фиксированная аналитика

Элемент отчета Фиксированная аналитика служит для отображения в отчете выбранных значений аналитических разрезов статей бюджетов, показателей бюджетов и нефинансовых показателей. Элемент в бюджетном отчете отображается всегда, вне зависимости от наличия значений по статьям и показателям бюджетов.

  • Вид аналитики - определяет значения каких справочников будут использоваться в качестве аналитических разрезов
  • Значение аналитики - выбранное значение аналитики, которое будет отображаться в отчете
  • Гиперссылка Дополнительные поля - открывает форму настройки дополнительных полей, отображаемых в бюджетном отчете
  • Комментарий - дополнительное описание, которое не будет отображаться в бюджетном отчете
