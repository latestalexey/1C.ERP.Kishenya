Настройка ячеек

Настройка ячеек таблицы - форма, позволяющая настроить значения ячеек сложной таблицы и установить дополнительное условное оформление.

  • Имя ячейки - имя текущей выделенной ячейки
  • Вид - вид показателя, отображаемого в ячейке
  • Вычисление - формула. Для производного показателя - формула вычисления, для статьи бюджета - формула заполнения.
  • Табличное поле представление отчета поддерживает различные сервисы для настройки значений ячейки: копирование, вставка и очистка значений ячеек.
