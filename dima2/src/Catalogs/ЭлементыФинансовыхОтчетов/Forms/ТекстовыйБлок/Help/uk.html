Текстовый блок

Элемент служит для отображения заголовков в бюджетных отчетах.
В текстовом поле указывается текст заголовка. Могут быть использованы дополнительные параметры заголовка, изменяющиеся в зависимости от параметров Бюджетного отчета или Экземпляра бюджета. Для добавления параметров используется подменю Добавить параметры. Состав параметров может быть расширен, если в виде бюджета указаны аналитические разрезы бюджета. Дополнительные аналитические разрезы бюджета так же доступны для вывода в заголовки.

  • Наименование - представление текстового блока в структуре элементов вида бюджета
  • Комментарий - вспомогательная информация, не влияющая на формирование бюджетов. Отображается в структуре элементов вида бюджета
  • Не выводить при заполнении бюджета - если флаг установлен, то при вводе документа Экземпляр бюджета система не будет отображать такой заголовок. Заголовок будет отображен при формировании Бюджетного отчета или при печати документа Экземпляр бюджета.
