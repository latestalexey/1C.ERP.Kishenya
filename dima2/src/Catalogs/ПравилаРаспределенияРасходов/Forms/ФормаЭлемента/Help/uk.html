Правило распределения расходов

Справочник предназначен для регистрации и настройки правил распределения материалов и работ, а также статей расходов по подразделениям и производственным этапам.

В зависимости от выбранного назначения правила доступны различные варианты базы распределения:

  • Распределение статей расходов по этапам производства - для данного назначения доступны автоматически рассчитываемые базы распределения:
    Количество (объем, вес) указанных материалов
    Количество (объем, вес) продукции
    Плановая стоимость продукции
    Количество работ указанных видов
    Сумма материальных затрат
    Сумма расходов на оплату труда
    Нормативная сумма расходов на оплату труда.
  • Распределение статей расходов по подразделениям - доступны базы распределения Сумма материальных затрат, Сумма расходов на оплату труда, а также возможно указание, что значение базы распределения Вводится вручную ежемесячно или Вводится вручную при изменении. При выборе последних двух вариантов список значений, определяющий базу распределения, вводится в отдельной форме при нажатии на гиперссылку Значения. При выборе варианта Вводится вручную при изменении значения вводятся только в случае их изменения, а при выборе Вводится вручную ежемесячно значения необходимо вводить ежемесячно.

Пример. Распределение расходов по подразделениям пропорционально объему потребленой электроэнергии подразделениями предприятия. Информация об объеме потребленной электроэнергии вводится ежемесячно. Для этого необходимо создать показатель для распределения расходов по подразделениям Объем потребленной электроэнергии, для которого установить, что он Вводится вручную ежемесячно. Таким образом, появляется возможность ежемесячно распределять расходы в соответствии с введенным текущим значением показателя.
  • Распределение материалов и работ - доступен следующий ряд баз распределения:
    Количество (объем, вес) указанных материалов
    Количество (объем, вес) продукции
    Плановая стоимость продукции
    Количество работ указанных видов
    Норматив расхода данного материала

Количество (объем, вес) указанных материалов

Для этой базы распределения существует возможность указать материалы или группы материалов, по которым будет строиться данная база. Для построения базы программа выбирает данные об использовании указанных материалов в производстве в течение месяца. Для базы По объему, весу количество материала умножается на коэффициент объема (веса) из карточки номенклатуры.

Количество (объем, вес) продукции

База распределения собирается из информации выпусков без распоряжений и информации о производимой продукции по заказам. При этом, учитывая, что производственный цикл по заказу может быть длительным (несколько месяцев), то корректным с экономической точки зрения является использование информации о выполненных этапах производства и плановом количестве выпуска по формуле:
Расчетное количество выпуска по заказу = Плановое количество выпуска(берется из заказа) *Количество выполненных этапов в текущем месяце / Общее количество запланированных этапов по заказу.

Количество работ указанных видов

База строится на основании данных об отнесении трудозатрат с указанными видами работ на выпускаемую в течение месяца продукцию

Плановая стоимость продукции

Данная база распределения строится аналогично базе Количество продукции, затем количество продукции умножается на плановую цену. Вид плановой цены устанавливается в Администрирование/Производство и ремонты/Плановые цены материалов, работ и отходов

Норматив расхода данного материала

Данные для базы выбираются аналогично как для базы Количество продукции, затем для выпусков без распоряжения норматив расхода распределяемого материала берется из спецификации, указанной в строках документа выпуска, а для выпусков по заказам норматив берется из спецификации, указанной в соответствующей строке табличной части Продукция Заказа на производство (или Заказа переработчику)

Сумма материальных затрат

Данные для базы выбираются из регистра накопления Партии незавершенного производства, учитывается суммовая оценка поступивших в течение месяца материальных затрат.

Сумма расходов на оплату труда

База формируется по данным регистров Трудозатраты незавершенного производства и Трудозатраты на выпуск без распоряжений, при этом берется фактическая сумма трудозатрат.

Нормативная сумма расходов на оплату труда

База строится аналогично базе Сумма расходов на оплату труда, но при этом для расчета берется нормативная стоимость, а не фактическая.

Для правил распределения статей расходов по подразделениям доступно указания направления распределения расходов. Выбор одного из вариантов определяет то, как расход будет распределяться по подразделениям.

Если в натройках Подразделения отключено использование документа Списания затрат на выпуск, то применяются следующие базы распределения: Количество продукции, Объем продукции, Вес продукции, Плановая стоимость продукции.

Также в элементе справочника можно установить отбор по группам (видам) продукции, который будет учитываться при построении базы распределения, и установить статью калькуляции, которая будет автоматически подставляться в документ распределения при выборе данного правила распределения
