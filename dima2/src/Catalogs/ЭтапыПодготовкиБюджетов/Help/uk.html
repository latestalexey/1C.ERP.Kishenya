Этапы подготовки бюджетов

Справочник предназначен для регистрации определенных Бюджетной моделью последовательности этапов с заданными порядком исполнения по датам начала работ, а так же срокам, установленным для составления и утверждения бюджетов и последующего проведения план-фактного анализа.

Этап подготовки бюджетов представляет собой задачу с указанным действием (например, ввод или проверка плана, анализ отклонений), планируемая на ответственного исполнителя с учетом параметров, настроенных для выбранного вида бюджета (подразделение, сценарий и т.п.).

Этапы бюджетного процесса группируются в Группы этапов, которые задают способ выполнения задач: последовательно или параллельно.

Группы этапов подготовки бюджетов являются повторяемыми и имеют настройку периодичности: неделя, месяц, квартал, полугодие, год, а так же настройку времени создания задач, например, за 5 календарных дней до начала периода или через 3 рабочих дня после завершения периода. Кроме того, группы могут быть вложены друг в друга и при этом содержать настройку периодичности, например, в годовой группе находятся группы по кварталам, а в них группы по месяцам.

Управление последовательностью создания бюджетов через этапы подготовки бюджетов позволяет обеспечить возможность использования данных одних бюджетов (или их сценариев) при формировании других бюджетов, взаимосвязанных между собой правилами заполнения.

Из рабочего места Монитор бюджетных процессов по настроенному плану подготовки бюджетов предусмотрено формирование отчета в виде диаграммы Ганта.
