Шаблоны ввода нефинансовых показателей

Шаблоны ввода нефинансовых показателей позволяют гибко задать структуру документа ввода показателей, установив группирующие отборы в шапку или строки документа Установка значений нефинансовых показателей, указав фиксированные значения аналитик в колонках или использовать для ввода несколько нефинансовых показателей.

Если шаблон явно для показателя не указан, то ввод осуществляется в режиме ввода по показателю, по автоматически настроенному для него шаблону. Причем в одном шаблоне нефинансовые показатели должны иметь совместимую аналитику.

Шаблоны настраиваются мастером по шагам-закладакам:

Выбор показателей

Выбор типа формы для ввода показателей

Выбор расположения полей в документе

Выбор способа заполнения

Выбор показателей


На первом шаге выбирается перечень нефинансовых показателей, для ввода которых создается шаблон. При этом помощник может помочь пользователю выбрать только совместимые нефинансовые показатели, т.е. показатели, имеющие одинаковые характер и детализацию. Наименование шаблона генерируется автоматически после выбора перечня нефинансовых показателей, для которых создается шаблон.

Выбор типа формы для ввода показателей


На втором шаге выбирается вариант размещения реквизитов формы для ввода значений нефинансовых показателей: все поля в шапке, в простой и сложной таблице. Полученный вариант формы документа Установка значений нефинансовых показателей можно просмотреть при помощи команды Просмотреть форму документа.

Выбор расположения полей в документе


На третьем шаге настраивается порядок размещения полей в таблице, в зависимости от выбранного варианта размещения.

Выбор способа заполнения


На последнем шаге можно настроить алгоритм заполнения значений нефинансовых показателей по данным других подсистем при помощи схемы компоновки данных. При этом могут использоваться предопределенные схемы прикладного решения или произвольные схемы, созданные самостоятельно.

По окончании работы помощника в информационной базе создается шаблон, который можно использовать в документе Установка значений нефинансовых показателей с видом операции Ввод значений по шаблону.
