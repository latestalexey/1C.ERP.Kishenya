Регистр сведений «Контрагенты включаемые в отчет о контролируемых операциях»

Регистр сведений содержит перечень контрагентов организации, включаемых в регламентированный отчет этой организации о контролируемых операциях. В регламентированный отчет о контролируемых операциях включаются только операции с контрагентами, указанными в регистре сведений. Операции с контрагентами, не включенными в регистр сведений, в регламентированный отчет не попадают, даже если операции с ними имеются в документе Сведения о контролируемых операциях.

Регистр сведений может быть заполнен автоматически по кнопке "Заполнить список", при этом в список попадают контрагенты, указанные в документе Сведения о контролируемых операциях, сумма операций с которыми в течение года составляет не менее 50 млн. грн.

Контрагентов можно добавлять в регистр сведений вручную, при этом необходимо указать для них "№ приложения отчета" при помощи кнопки "Восстановить нумерацию приложений".
