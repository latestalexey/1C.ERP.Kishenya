Настройка подсистемы

Для настройки подсистемы в разделе НСИ и администрирование - Бюджетирование и планирование необходимо осуществить следующие настройки:

  • Для возможности использования функционала по финансовому планированию установить функциональную опцию Бюджетирование.
  • Для возможности управления регламентом подготовки и контроля своевременности исполнения бюджетов ответственными лицами установить функциональную опцию Использовать оповещения для задач бюджетирования.
  • Для возможности автоматического расчета хранимых данных бюджетирования в момент проведения документов (формирования отчетов) установить функциональные опции Автоматически запускать фоновое задание расчета хранимых данных при проведении документов и Автоматически запускать фоновое задание расчета хранимых данных из отчетов.

Настройка НСИ

Начальная работа с подсистемой Бюджетирования осуществляется по средствам подготовки нормативно-справочной информации: Статей бюджетов, Видов бюджетов, Сценариев, а так же используемых в них аналитик.

На основании заполненых в справочниках данных осуществляется создание Модели бюджетирования, которая в свою очередь определяет:

  • Использование основных аналитик бюджетов: Организации и Подразделения.
  • Правила заполнения плановых показателей и статей.
  • Этапы бюджетного процесса, задающие регламент планирования и контроля по используемым в модели видам бюджетов.
