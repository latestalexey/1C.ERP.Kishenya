Бюджеты

Предназначены для возможности финансового планирования процессов: разработки, исполнения, контроля и анализа финансового плана, учета и контроля доходов и расходов, получаемых предприятием на всех уровнях управления. В общем случае бюджеты можно сгруппировать следующим образом:

  • основные бюджеты (бюджет доходов и расходов, бюджет движения денежных средств, прогнозный баланс);
  • операционные бюджеты (бюджет продаж, бюджет производства, бюджет прямых материальных затрат, прямых трудовых затрат и т.д.);
  • вспомогательные бюджеты (план капитальных вложений, кредитный план, бюджеты налогов);
  • дополнительные (специальные) бюджеты (бюджет распределения прибыли, планы отдельных проектов и программ).

Особо следует выделить основные виды бюджетов, так как они являются ключевыми итоговыми формами бюджетного процесса, тесно взаимосвязаны между собой и формируются на основании данных остальных видов бюджетов. При создании основных бюджетов необходимо соблюдать определенную последовательность работы по их формированию, которая регламентируется внутри системы через настройку Этапов бюджетного процесса по бюджетной модели.

Для ввода плановых значений по выбранному Виду бюджета за определенный период в разрезе Сценария и Модели бюджетирования используются Экземпляры бюджета. Экземпляр бюджета формируется в виде отчета при заполнении значений строк автоматически пересчитывает итоги по группам статей, связанным статьям и показателям.
Для отчетов, формируемых по видам бюджетов используется рабочее место Бюджетные отчеты. Отчеты могут быть трех типов:

  • Бюджет – выводит только плановые значения;
  • Сравнение сценариев – позволяет сравнивать разные планы и видеть отклонения по статьям;
  • План-фактный анализ – позволяет сравнивать выбранный сценарий бюджетирования с фактическими данными, полученными за период.

Существует возможность изменять варианты бюджетных отчетов, добавляя в них новые группировки. Сохраненные варианты отчетов, при этом, остаются доступными для выбора из рабочего места.
