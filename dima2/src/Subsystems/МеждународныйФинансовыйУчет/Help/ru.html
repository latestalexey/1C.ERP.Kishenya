Международный финансовый учет

Назначение подсистемы

Подсистема Международный финансовый учет позволяет решать следующие задачи:

  • Отражение документов оперативного учета в финансовом учете;
  • Подготовка отчетности в соответствии с принципами международных стандартов финансовой отчетности (МСФО), включая возможность настройки дополнительных отчетов для проверки и анализа учетных данных;
  • Подготовка отчетности по правилам управленческого учета, построенным на основе МСФО;
  • Гибкая настройка форм финансовой отчетности.

Настройка подсистемы

Для настройки подсистемы в разделе НСИ и администрирование – Настройка разделов – Международный финансовый учет необходимо установить функциональную опцию Международный финансовый учет.

Работа подсистемы

Работа в подсистеме начинается с настройки международного плана счетов в разделе Настройки и справочники - План счетов "Международный". План счетов может быть как частью поставляемой модели, так и настраиваться в соответствии с планом счетов, принятым на предприятии (рекомендуемым аудиторами).

Далее, на основе принятой на предприятии учетной политики, необходимо создать Учетную политику международного учета, в рамках которой будет проводиться настройка типовых проводок для Хозяйственных операций или настройка соответстия счетов и оборотов регламентированному плану счетов.

Настройка типовых проводок для хозяйственных операций заключается в настройке Шаблонов проводок в разрезе выбранной учетной политики. Для использования механизма шаблонов проводок необходимо включить функциональную опцию НСИ и администрирование – Международный финансовый учет – Проводки по данным оперативного учета.
Настройка шаблонов происходит в рабочем месте Настройка шаблонов проводок (Международный финансовый учет - Настройки и справочники - Настройка формирования проводок по данным опер. учета), работа в котором может осуществляться в двух режимах:

  • По хозяйственным операциям - создание и редактирование шаблонов проводок;
  • По плану счетов - просмотр настроенных шаблонов и уточнений счетов в группах финансового учета.


Для использования механизма трансляции операций из регламентированного учета в международный необходимо включить функциональную опцию НСИ и администрирование – Международный финансовый учет – Проводки по данным регламентированного учета. Далее в рабочем месте Настройка соответствия счетов и оборотов (Международный финансовый учет – Настройки и справочники – Настройка формирования проводок по данным регл. учета) задаются соответствия счетов регламентированного учета счетам международного учета и соответствие оборотов (проводок с указанной корреспонденцией счетов) регламентированного учета оборотам международного учета.

Для использования документов параллельного учета основных средств и нематериальных активов необходимо включить опцию НСИ и администрирование – Международный финансовый учет – Учет внеоборотных активов.

После настройки механизмов формирования проводок международного учета осуществляется настройка структуры отчетности в генераторе финансовых отчетов. Генератор финансовых отчетов представляет собой инструмент подготовки форм финансовых отчетов, источником данных для которых служат проводки по плану счетов. Настройка комплекта форм финансовой отчетности заключается в создании иерархической структуры показателей отчетов и состоит из следующих этапов:

  • Создание новых (или изменение поставляемых моделью) Видов финансовых отчетов;
  • Уточнение структуры показателей отчетности;
  • Объединение отчетов в Комплекты финансовых отчетов.


На этом этапе инициализационная настройка подсистемы заканчивается и возможен переход к основной работе по получению финансовой отчетности. В течение каждого отчетного периода, после того как ввод в систему и проверка документов оперативного учета завершены, по ним выполняется автоматическое формирование проводок международного учета из рабочего места Отражение документов в международном учете.

Ручной ввод бухгалтерских проводок осуществляется с использованием отдельных документов Операции (международный учет). Возможны следующие способы заполнения документа:

  • Ручной ввод корреспонденции счетов и суммы проводки;
  • Загрузка данных из табличного документа, например из таблицы Excel;
  • Использование ранее сохраненной типовой операции.


Контроль ведения учета осуществляется с помощью основных и дополнительных отчетов подсистемы: оборотно-сальдовых ведомостей, карточек счетов и т.п. В конце отчетного периода производится обновление нефинансовых показателей на их актуальные значения.

После завершения отражения всех операций в учете проводится формирование финансовой отчетности в соответствии с настроенным комплектом отчетов.

Анализ данных

Для контроля и анализа учетных данных в подсистеме предусмотрены различные виды отчетов.

Стандартные отчеты финансового учета включают в себя:

  • Анализ субконто (международный учет);
  • Анализ счета (международный учет);
  • Карточка счета (международный учет);
  • Сводные проводки (международный учет);
  • Оборотно-сальдовая ведомость по счету (международный учет);
  • Оборотно-сальдовая ведомость. Пробный баланс (международный учет).


Дополнительные отчеты для анализа и сверки учетных данных включают в себя:

  • Анализ денежных потоков;
  • Анализ взаиморасчетов контрагентов по договорам (международный учет);
  • Шахматная ведомость (международный учет);
  • Сверка оборотов международного и регламентированного учетов.

Список разделов и их назначение

Отражение документов в международном учете. Представляет собой рабочее место бухгалтера, предназначенное для автоматического формирования проводок в международном финансовом учете.

Финансовая отчетность. В подсистеме предусмотрена возможность гибкой настройки форм финансовой отчетности. Процесс настройки формы отчета заключается в создании его структуры из доступных элементов оформления отчета, группировок и показателей.
