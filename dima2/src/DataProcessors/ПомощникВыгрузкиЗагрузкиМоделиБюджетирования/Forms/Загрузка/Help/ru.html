Помощник выгрузки и загрузки модели бюджетирования

Помощник выгрузки/загрузки модели бюджетирования предназначен для обеспечения обмена данными бюджетной модели между несколькими информационными базами.

Инструменты реализации задачи обмена:

  • Выгрузка модели бюджетирования в файл для обмена с другими информационными базами;
  • Загрузка модели бюджетирования из файла, содержащего данные загружаемой модели бюдежетирования;
  • Копирование модели бюджетирования в рамках одной информационной базы.

Помощник загрузки модели бюджетирования обеспечивает загрузку модели бюджетирования из файла обмена, который содержит выгруженную ранее модель из другой информационной базы.

При загрузке файла можно обновить существующую модель бюджетирования или создать новую.

Загрузке подлежат следующие объекты:

  • Модель бюджетирования;
  • Виды бюджетов;
  • Этапы бюджетного процесса;
  • Правила лимитирования расхода ДС.


Наряду с перечисленными объектами загружаются бюджетные статьи, показатели и аналитики бюджетирования. Причем новые объекты создаются, а найденные не подлежат обновлению.
