Ввод факта по маршрутному листу

Используется для ввода фактического расхода материалов, выпуска изделий и трудозатрат при выполнении производственной операции маршрутного листа. Заполнение возможно только для данных у которых в Маршуртном листе указана конкретная операция.

В форме необходимо заполнить поля Факт или Отклонение, а для трудозатрат уточнить бригады.
Введенные данные переносятся в маршрутный лист.
