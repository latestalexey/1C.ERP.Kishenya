Настройка шаблонов проводок (международный учет)

Рабочее место предназначенно для настройки бухгалтером шаблонов проводок в Международном финансовом учете.

Хозяйственные операции учитываются в оперативном контуре в аналитических регистрах накопления, которые предназначены для отражения движений между сущностями учета (контрагенты, номенклатура, денежные средства, доходы/расходы, прочие активы/пассивы) и хранения данных с необходимой детализацией.
Проводки МФУ формируются автоматически по данным хозяйственных операций, хранимых в аналитических регистрах, на основании настроенных шаблонов.

Для настройки соответствия счетов и оборотов регламентированного учета счетам международного учета используется другое рабочее место - Настройка соответствия счетов и оборотов (Международный финансовый учет – Настройки и справочники – Настройка формирования проводок по данным регл. учета), которое становится доступным после включения функциональной опции НСИ и администрирование – Международный финансовый учет – Управление параметрами международного финансового учета - Проводки по данным регламентированного учета.

Работа в рабочем месте Настройка шаблонов проводок возможна в двух режимах:

По хозяйственным операциям.

По плану счетов

По хозяйственным операциям.

Режим используется для основной настройки Шаблонов проводок в зависимости от выбранной Хозяйственной операции. Кнопка Изменить отражение в учетной политике позволяет включить/исключать шаблон проводки из выбранной учетной политики.

Шаблоны проводок можно объединять в группы совместного применения. Доступны следующие виды групп:

  • Вытеснение - применяется первый в порядке следования шаблон проводки, удовлетворяющий условиям дополнительного отбора.
  • Все - примененяются все шаблоны проводок, входящие в группу.

Для изменения порядка следования шаблонов в группе необходимо воспользоваться командами Переместить вверх или Переместить вниз.

По плану счетов

Режим используется для просмотра созданных по хозяйственным операциям шаблонов проводок, проверки и изменения (дополнения) настроек существующих шаблонов, с возможностью настройки шаблонов по замкнутой цепочке процессов, в которых задействован счет МФУ, например, «продажа - оплата - возвраты».

Процесс настройки состоит из этапов:

  • формирование списка шаблонов проводок по хозяйственным операциям;
  • проверка целостности (сбалансированности) настройки счетов, используемых в шаблоне по умолчанию;
  • проверка счетов, уточняемых в шаблоне с помощью Группы финансового учета и других настроек.
