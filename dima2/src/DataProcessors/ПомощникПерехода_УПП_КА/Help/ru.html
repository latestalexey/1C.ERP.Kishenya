Помощник перехода с УПП, ред. 1.3

Обработка Помощник перехода предназначена для пользователей программы Управление производственным предприятием для Украины, редакция 1.3 (УПП 1.3), которые переходят на работу в программе.

В результате переноса данных пользователи получают возможность начать работать с теми данными, с которыми они прежде работали в УПП 1.3: это вся нормативно-справочная информация, а также информация об остатках по всем разделам учета на определенную дату.

Подробное описание методики перехода содержится в руководстве "Руководство по переходу с УПП 1.3.htm", входящем в комплект поставки.
