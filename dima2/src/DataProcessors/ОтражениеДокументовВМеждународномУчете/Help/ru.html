Отражение документов в международном учете

Рабочее место предназначенно для автоматического формирования бухгалтером проводок в Международном финансовом учете.
Формирование проводок в международный учет по документам оперативного учета производится отложено на периодической основе (документ при проведении проводок не формирует, а попадает в список ожидаемых для последующего отражения в международном учете).

Из рабочего места доступны функции:

Просмотр документов

Контроль проводок

Настройка автоматического отражения документов в учете по расписанию.

Просмотр документов
  • ожидающих отражения в международном учете;
  • в том числе в закрытом для изменений периоде;
  • требующих настройки шаблонов проводок;
  • отраженных в международном учете;
  • отраженных вручную.

Контроль проводок

Контроль настроеных на текущий момент времени для выбранной организации учетных политик с целью выявления в них ошибок, связанных с неправильной настройкой шаблонов проводок.

Настройка автоматического отражения документов в учете по расписанию.

В режиме автоматического отражения документов в учете, алгоритм формирования проводок выводит список документов, в которых выбраны Хозяйственные операции с не созданными для них или требующими исправления Шаблонами проводок.

Из списка документов возможны следующие действия:

  • переход в форму Настройка шаблонов проводок,
  • переход в форму Настройка соответстивя счетов и оборотов из регламентированного учета.
