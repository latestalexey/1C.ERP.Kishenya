Настройка отражения документов в регл учете

Рабочее место предназначено для настройки отражения операций документов в проводках регламентированного учета и для непосредственного отражения документов в соответствии с указанныи настройками.

Рабочее место позволяет настроить порядок отражения документов в регламентированном учете заранее вручную, либо указывать необходимые счета по мере появления новых операций. В последнем случае новые строки для отражения будут добавлены автоматически и счета, необходимые для указания, будут выделены.

Настройка осуществляется в разрезе:

  • аналитик учета, например статьи расходов.
  • мест учета, например подразделения предприятия.

Не рекомендуется изменять счета учета для уже настроенных операций. Следует понимать, что подобное изменение затронет не только новые документы, но и уже отраженнные в регламентированном учете, если для них потребуется переоформление проводок, например при перепроведении. В этом случае счета будут установлены в соответствии с новой настройкой.

Механизм формирования проводок построен таким образом, что для каждой отдельной операции не требуется указание счетов учета и значений субконто. Для этого достаточно один раз настроить порядок отражения таких операций.
Проведение документа и отражение этого документа на счетах регламентированного учета разнесено по времени, т.е., если для какой-то операции еще не настроено отражение в регламентированном учете, это не помешает проведению документа и формированию движений по другим системам учета.

Основное рабочее место по отражению документов в регламентированном учете состоит из следующих групп данных:

  • Отбор документов - позволяет отобрать документы к отражению по определенной организации и сформированных не позже указанной даты.
  • Проверка документов - показывает количество проверенных, непроверенных и требующих повторной проверки документов.
  • Формирование проводок по документам - показывает количество документов ожидающих отражения, отраженных в регл. учете и не отраженных из-за ошибок.
  • Ручное изменение проводок документов - показывает количество актуальных корректировок документов операциями регламентированного учета или ожидающих подтверждения (если проводки документов, на основании которых вводилась операция были изменены), а так же количество вручную отраженных и требующих подтверждения ручного отражения документов.
  • Состояние счетов учета - указывает на необходимость настройки счетов и позволяет открыть соответствующую форму по кнопке Настроить счета учета.
  • Отражение документов в регл. учете - позволяет отразить документы в регл. учете по нажатию одноименной кнопки. Будут отражены все документы, в соответствии с выставленными отборами, которые ожидают отражения, либо не были отражены из-за ошибок. Здесь же можно настроить автоматическое расписание, по которому будет вызываться процедура отражения документов в регл. учете.

Для каждой из групп документов по соответствующим гиперссылкам можно открыть список указанных документов.

Отражение документов в регламентированном учете может осуществляться по настроенному расписанию при помощи регламентного задания. Для его активации и настройки необходимо установить флаг Автоматическое отражение документов в регл. учете по расписанию.

Группа проверки документов, а так же подгруппы

  • требующих повторной проверки (группы Проверки),
  • вручную отраженных документов (группы Ручного изменения проводок),
  • документов требующих подтверждения ручного отражения (группы Ручного изменения проводок).

Показываются в зависимости от настроек соответствующих функциональных опций.
