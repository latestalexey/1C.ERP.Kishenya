Настройка соответствия счетов и оборотов

В рабочем месте Настройка соответствия счетов и оборотов задаются соответствия счетов регламентированного учета счетам международного учета и соответствие оборотов (проводок с указанной корреспонденцией счетов) регламентированного учета оборотам международного учета.

Приоритет правил соответствия оборотов выше, чем правила соответствия счетов, то есть к проводке, для которой задано соответствие оборотов не будут применяться правила соответствия счетов.

Отражение документов в международном учете учитывает оба способа формирования проводок: по шаблонам проводок для оперативного учета и по правилам трансляции из регламентированного учета. Во избежание конфликтных ситуаций, при использовании сразу двух режимов, необходимо настроить игнорирование хозяйственных операций для оперативного учета и игнорирование счетов и оборотов регламентированного учета. В противном случае возможно задвоение проводок при отражении проводок в международном учете.

При использовании трансляции операций из регламентированного учета требуется соблюдение последовательности отражения - сперва в регламентированном учете, потом в международном, иначе документы будут помечаться как ожидающие отражения по регламентированному учету и отразятся в международном учете.

Для указания соответствия счетов существует возможность настроить дополнительные отборы, применяемые к проводке регламентированного учета. Для счета международного учета настраивается соответствие субконто. Если субконто не однозначно соответствует регламентированному учету, то можно выбрать его из атрибутов субкотно регламентированного учета. Например, в регламентированном учете субконто номенклатура, а в международном учете для соответствующего счета – это группа номенклатуры. В этом случае настраивается заполнение субконто из атрибута номенклатуры.

Настроенные соответствия счетов не разделяются на обороты по дебету или кредиту и применяются к счету, независимо от его расположения в проводке. Правила соответствия счетов настраиваются в разрезе учетной политики, аналогично шаблонам проводок из оперативного учета.

Существует возможность установки признака игнорировать счет регламентированного учета для отражения в международном учете. При использовании данной настройки в учете не возникает ошибок, связанных с отсутствием настройки соответствия оборотов. Документы с такими проводками будут считаться отраженными в оперативном учете, но проводки по международному учету для них не возникнет.
Настройка правил трансляции оборотов проводится на закладке Соответствие оборотов.

Шаблон проводки по оборотам регламентированного учета состоит из указания корреспонденции счетов регламентированного учета и соответствующей ей корреспонденции счетов международного учета. Существует возможность указания дополнительных отборов и соответствия субконто, аналогично настройкам счетов.

Для оборотов можно указать ручное уточнение проводки. В этом случае документы при отражении будут попадать в список документов, требующих ручное уточнение проводок, аналогично ручному уточнению для шаблонов проводок из оперативного учета.
Существует возможность настройки игнорирования оборота, для этого используется отметка Не отражается в международном учете.

Для использования механизма шаблонов проводок, формируемых по данным оперативного учета, необходимо включить функциональную опцию НСИ и администрирование – Международный финансовый учет – Управление параметрами международного финансового учета - Проводки по данным оперативного учета.
Настройка шаблонов происходит в другом рабочем месте - Настройка шаблонов проводок (Международный финансовый учет - Настройки и справочники - Настройка формирования проводок по данным опер. учета).
