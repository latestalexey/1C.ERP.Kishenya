Настройка значений показателей для распределения расходов

Рабочее место предназначено для настройки значений показателей для распределения расходов.

Обработка работает в двух режимах:

  • Просмотр и изменение значения конкретного показателя для нескольких подразделений. Обработка вызывается из справочника Показатели для распределения расходов.
  • Просмотр и изменение значений ряда показателей для конкретного подразделения. Обработка вызывается соответствующей командой в панели навигации карточки подразделения.
