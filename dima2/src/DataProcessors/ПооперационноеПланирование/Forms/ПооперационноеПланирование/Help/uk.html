Пооперационное планирование

Рабочее место предназначено для формирования пооперационного расписания производства.
При планировании выполняется размещение операций маршрутных листов, принадлежащих подразделениям с пооперационной методикой управления. Операции размещаются в расписании с даты начала выполнения маршрутного листа.

Расчет расписания


Для расчета расписания используется команда Рассчитать, которая открывает форму настройки параметров расчета.

Запись расписания


Для записи построенного расписания служит команда Записать.

Если в параметрах расчета расписания выбрано несколько моделей планирования, то при расчете будет построено несколько вариантов расписания, по количеству выбранных моделей. Для записи доступен лишь один из построенных вариантов расписания – оптимальная модель. Для выбора оптимальной модели следует выполнить двойной клик на соответствующем элементе диаграммы и выбрать команду Установить в качестве оптимальной модели. Оптимальная модель отображается в расписании жирным шрифтом.

В параметрах расчета расписания может быть указан сценарий пооперационного планирования. Расписание, построенное с использованием сценария, предназначено исключительно для анализа эффективности данного сценария, и не доступно для записи.

Анализ расписания


Для анализа рассчитанного расписания до его записи служит команда Анализ расписания, которая открывает отчет Анализ пооперационного расписания.
