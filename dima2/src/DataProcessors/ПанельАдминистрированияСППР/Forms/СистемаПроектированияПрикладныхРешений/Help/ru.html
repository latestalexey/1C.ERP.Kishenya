Система проектирования прикладных решений

Используя эту панель можно настроить интеграцию с Системой проектирования прикладных решений.
Интеграция с СППР предоставляет навигацию по функциональной модели конфигурации - для перехода к навигации используется команда Функциональная модель, которая располагается во всех формах рабочих мест.

Интеграция с СППР

Интеграция с СППР
  • URL - адрес Веб-сервиса СППР;
  • Пользователь, пароль - параметры аутентификации;
  • Проект - проект, в рамках которого получаются данные из СППР. Если указаны параметры соединения с СППР, то с помощью кнопки выбора можно выбрать проект из списка проектов СППР;
  • Проверить соединение - позволяет проверить соединение с Веб-сервисом СППР.
