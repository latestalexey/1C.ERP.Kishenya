Дерево ресурсных спецификаций

Форма Дерево ресурсных спецификаций предназначена для наглядного изображения структуры изделия. Может быть построено с учетом спецификаций производимых в процессе полуфабрикатов.

Этапы

Список этапов производства. Для кажого этапа отображается выходные изделия, потребляемые материалы и виды работ, требуемые для его выполнения. Изделие, для которого строится дерево спецификаций, является выходным изделием одного из этапов.

Выходные изделия

Список изделий, которые производятся на этапе производства.

Материалы и услуги

Список материалов и услуг, которые потребляются на этапе производства. Полуфабрикатами являются материалы, для которых в ресурсной специйфикации установлен признак Производится в процессе. Спецификации полуфабрикатов так же отображается в виде дерева.

Трудозатраты

Список видов работ, которые требуется выполнить на этапе производства.
