Формирование заказов на производство по плану

Рабочее место предназначено для формирования заказов на производство продукции в количестве, зафиксированном планами производства. Рабочее место позволяет формировать заказы за один сеанс, а также выборочно, по отдельным позициям номенклатуры на заданный период работы производственных подразделений.

В таблице отображается номенклатура, которую необходимо заказать. Для заполнения таблицы используется команда Заполнить. Для каждой номенклатуры в таблице система предложит необходимое количество в колонке К заказу.
Если заказы на производство формируются на Период, включающий несколько периодов, согласно периодичности сценария, то общее количество К заказу для позиции номенклатуры будет поделено между периодами, согласно сценария.

Пример. Допустим сценарий планирования имеет периодичность Месяц. Необходимо создать заказы на производство на два месяца вперед. В поле Период с указываем период продолжительностью два месяца и заполняем таблицу. Для каждой номенлкатуры в таблице будет две строки: первая - отражает запланированные (планами производства) количества первого месяца, вторая - второго месяца.

После заполнения таблицы можно откорректировать количество к заказу и установить отметки напротив строк, которые необходимо включить в формируемый заказ на производство.
Команда Создать заказ создает заказ на производство, заполняет и выполняет проведение заказа.

Количество рекомендуемое к заказу, рассчитывается на основании количества, зафиксированного на период в планах производства за вычетом ранее созданных заказов на производство за этот же период. При заполнении поля По подразделению, учитываются только планы и заказы указанного подразделения.
