Денежные документы

Журнал содержит информацию по имеющимся денежным документам в организациях, а ткже список всех учетных документов

Журнал денежных документов содержит две закладки:

  • Денежные документы в наличии. На этой закладке отображается инфомрация по имеющимся денежным документам в разных организациях предприятия
  • Движение денежных документов. На этой закладке отображается список учетных документов.


На закладке Денежные документы в наличии выводятся все денежные документы, имеющиеся на предприятии в разрезе организаций, подразделений, материально-ответственных лиц, денежных документов.

Оформить поступление нового денежного документа можно при помощи кнопки Создать, будут доступны возоможности сформировать Поступление денежных документов от поставщика и Поступление денежных документов от подотчетного лица

Оформить выбытие денежных документов можно при помощи кнопки Оформить, будут доступны возможности оформить Выдачу подотчетному лицу, Возврат поставщику, Списание на расходы/активы, Перемещение между МОЛ.

На закладке Движение денежных документов отображаются учетные документы по поступлению, выдаче, возврату, перемещению денежных документов, списанию на расходы. Оформление новых документов производится по кнопке Создать.

Журнал денежных документов имеет возможности по фильтрации денежных документов по органиазции, подразделению, материально-ответственному лицу.
