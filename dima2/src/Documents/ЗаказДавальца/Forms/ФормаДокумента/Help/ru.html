Заказ давальца

Заказ давальца - это запрос клиента на оказание услуг по переработке принадлежащего ему сырья и материалов, именуемых давальческим сырьём. Права на сырье и материалы, полученные в переработку, на всех этапах переработки остаются за давальцем. Продукция, выпущенная с использованием давальческого сырья, так же принадлежит давальцу. При производстве давальческой продукции допустимо использование собственных материалов.

За исключением некоторых специфических операций, правила учета взаимоотношений с давальцем и покупателем схожи, поэтому общая функциональность заказов клиентов применима и к заказам давальцев.

Основные данные

Давалец является Клиентом организации, поэтому при выборе давальца список контрагентов фильтруется по соответствующему типу взаимоотношений. Заказ давальца может формироваться с указанием договора, который определяет порядок расчетов: по заказам, накладным или договорам. В случае ведения расчетов по накладным долг давальца образуется при оформлении отчета давальцу, а не при передаче готовой продукции. Если договор не указан, то объектом расчета выступает заказ давальца, взаиморасчеты ведутся по заказам.

Поскольку поставка материалов и отгрузка готовой продукции может осуществляться с указанием разных складов, в документе необходимо заполнить 2 склада: Склад (продукция) и Склад (материалы). Для указания складов в табличной части необходимо включить функциональные опции Реализация товаров с нескольких складов в разделе НСИ и администрирование - Продажи - Использование заказов и/или Поступление товаров на несколько складов в разделе НСИ и администрирование - Закупки - Условия и порядок оплаты соответственно.

В группе Услуга по переработке указывается номенклатура с типом Услуга, которая будет определять значения по умолчанию Ставки НДС и Содержания услуги по переработке.

Если в шапке документа указать Вид цены, то при добавлении новых строк выпускаемой продукции, цена будет заполняться автоматически. В заказе давальца для выбора доступны виды цен с установленным признаком Использовать при передаче продукции давальцу и с признаком Цена включает НДС соответствующим заказу.

Для заказов давальцев доступно указание приоритета, который может использоваться для выделения и сортировки документов в списке.

Если объектом расчета является заказ или договор, то в правой части подвала отображается информация об этапах и сроках оплаты услуг по переработке. При нажатии на ссылку открывается форма редактирования графика оплаты. В правой части отображается информация о поступивших оплатах по заказу или договору. При нажатии на ссылку открывается карточка расчетов с клиентом.

В правой части подвала отображается текущее состояние заказа давальца. При нажатии на гиперссылку открывается отчет Состояние выполнения документов.

Выпускаемая продукция

Для каждой строки табличной части Выпускаемая продукция заполняется номенклатурная позиция, которая будет произведена из сырья заказчика, а также стоимость выпуска этой продукции. Стоимость выпуска позиции заказа давальца складывается из стоимости собственных материалов и стоимости услуги по переработки.

Собственные материалы и услуги по переработке оцениваются по виду цен, указанному в табличной части или на закладке Основное. Для удобства оформления заказов давальцев рекомендуется в рамках одного вида цены, задать цены на услуги по переработке и собственные материалы, которые расходуются при производстве продукции. Заполнение стоимости собственных материалов выполняется по команде Заполнить - Стоимость собственных материалов. Расчет стоимости выполняется, если для продукции указана ресурсная спецификация.

Для детализации стоимости переработки предусмотрен отчет Расшифровка стоимости переработки, который разворачивает по спецификации собственные и давальческие материалы, требуемые для производства.

Кроме расчета стоимости собственных материалов спецификация используется для заполнения списка материалов и подстановки в заказ на производство и выпуск продукции без заказа на производство при вводе на основании.

Отгрузка продукции по заказу возможна после установки действия по обеспечению Отгрузить обособленно в строках табличной части и статуса документа К отгрузке. При установке статуса становится обязательным для заполнения поле Дата отгрузки. Его можно указать как для всего документа в целом, так и для каждой строки.

Отмена строк и закрытие частично исполненного заказа давальца выполняется аналогично заказу клиента: с указанием причины отмены.

Сырье и материалы для производства

Оказание услуг по переработке возможно как с использованием исключительно давальческого сырья, так и с частичным использованием собственного сырья и материалов (в этом случае их стоимость учитывается в стоимости услуг по переработке). В табличной части Сырье и материалы для производства указывается только давальческое сырье. Также может быть заполнена Залоговая стоимость, получаемого сырья.

Если использовать давальческое сырье не планируется и производство продукции планируется полностью из собственных материалов, то в этом случае рекомендуется использовать документ Заказ клиента.

При необходимости произвести обособленное резервирование товара со свободных остатков склада из собственных материалов или изъять давальческое сырье или выпущенную продукцию (полуфабрикат) другого заказа в пользу текущего, можно воспользоваться командой Зарезервировать под назначение.

Для заполнения списка материалов доступны следующие команды:

  • Заполнить по спецификации - заполнение по спецификации производимой продукции. Спецификация должна быть указана на закладке Выпускаемая продукция.
  • Заполнить по фактической потребности - заполнение по данным спецификации заказа на производство.

Список сырья и материалов может быть дополнен возвратной тарой. Тара не является давальческим материалом, и может быть выкуплена у давальца, возвращена давальцу или оприходована как собственный товар. Если тара подлежит возврату, то на закладке Дополнительно следует установить признак Вернуть многооборотную тару. Возврат осуществляется документом Возврат сырья давальцу, выкуп - документом Выкуп возвратной тары. Если тара не подлежит возврату, то при поступлении оформляется оприходование тары как собственного товара по указанной в документе стоимости.

Дополнительно

На закладке Дополнительно можно указать признак Цена включает НДС, Валюту и другие параметры:

  • Желаемая дата отгрузки и Желаемая дата поступления учитываются при формировании плановых дат отгрузки и поступления.
  • Сделка, в рамках которой сформирован заказ давальца. Все документы по обслуживанию заказа давальца могут быть прикреплены к сделке.
  • Подразделение диспетчирующее производство продукции по заказу давальца.
