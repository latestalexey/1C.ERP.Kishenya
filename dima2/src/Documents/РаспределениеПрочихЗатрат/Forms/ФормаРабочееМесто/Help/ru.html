Форма рабочее место

Рабочее место предназначено для отнесения расходов, накопленных в течение месяца на себестоимость продукции, произведенной в течение данного периода.
С помощью рабочего места распределяются только те расходы, которые накоплены по статье расходов с вариантом распределения на производственные затраты.
