Распределение материалов и работ

Документ предназначен для распределения материалов и работ, списанных в производство, но не потребленных непосредственно при выполнении этапов производства. При проведении документа производится распределение материалов и работ, которые не были использованы в производственном процессе, на себестоимость производимой продукции, а также списание на статьи расходов.

В документе распределения указывается информация о количестве материала, который остается в Подразделении (поле В наличии), количестве материала, который необходимо распределить по правилу распределения, вручную отнести на этапы заказов, выпуски без распоряжений, и списать на статьи расходов.

Для настройки распределения доступны следующие закладки документа:

Основное

По правилу распределения

На этапы заказов

На выпуски без распоряжений

На статьи расходов

Основное

Содержит основные реквизиты документа, такие как нмер, дата, домментарий, информацию о распределяемом материале.

По правилу распределения

Содержит настройки распределения указанного количества материалов согласно выбранному правилу распределения. При выборе правила распределения, его настройки копируются в документ и далее пользователь может их самостоятельно изменить в случае необходимости. Также есть возможность перенести введенные в документе настройки в выбранное правило распределения с помощью соответствующей кнопки на закладке По правилам. В зависимости от выбранного правила распределения, есть возможность сформировать контекстный отчет, вызываемый по гиперссылке Проверить наличие данных для базы распределения. Отчет показывает наличие данных, необходимых для формирования базы распределения.

На этапы заказов

Содержит информацию о этапах заказа на производство и заказа переработчику, на которые нужно отнести указанное количество распределяемого материала.

На выпуски без распоряжений

На данной вкладке указывается информация о документах выпуска продукции без распоряжения, на которые нужно отнести указанное количество распределяемого материала.

На статьи расходов

Позволяет ввести статьи расходов и аналитику расходов, на которые нужно отнести распределяемый материал в указанном количестве.

В заголовке закладок по мере заполнения документа показывается информация о количестве материала (работы), распределенного в соответствии с настройками данной закладки.
