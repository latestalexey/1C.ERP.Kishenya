Корректировки обособленного учета запасов

Документ предназначен для корректировки данных при ведении обособленного учета по сделкам, менеджерам или подразделениям. Ведение обособленного учета позволяет детализировать данные о поступлениях и расходах в рамках объектов обособленного учета (сделок, менеджеров и т.д.). Обособленный учет ведется только в том случае, если в разделе Администрирование/Финансовый результат установлен флажок Обособленный учет себестоимости товаров по видам запасов и выбран объект для ведения обособленного учета: По подразделениям или менеджерам или По сделкам.

Как зарегистрировать новую корректировку обособленного учета?

Для того чтобы зарегистрировать новую корректировку обособленного учета необходимо в списке корректировок воспользоваться командой Создать.
