Документ "Регистр учета"

Документ предназначен для хранения сформированного бухгалтерского отчета заверенного с помощью ЭЦП, в базе данных. С возможностью последующего чтения и печати.
Документ создается автоматически при сохранении бухгалтерского отчета. Реквизиты регистра учета недоступны для редактирования.

В документе доступны следующие команды:

Открыть - открыть сохраненный регистр учета в том формате, в котором он был сохранен;

Подписать ЭЦП - подписать сохраненный файл электронной подписью;

Сохранить - сохранить на диск регистр учета, подписанный электронной подписью.
