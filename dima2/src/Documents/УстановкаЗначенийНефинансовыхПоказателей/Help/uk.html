Значения нефинансовых показателей

Документ предназначен для установки значений нефинансовых показателей бюджетов вручную или заполнением по настроенным правилам. Документ позволяет вводить значения по одному показателю или использовать шаблон, где можно настроить структуру документа из аналитики, периодов и показателей в удобном для массового ввода виде.
