Амортизация и износ основных средств

Документ предназначен для отражения в системе операций по расчету амортизации основных средств. Для избежания повторного начисления из состава активов исключаются объекты и по которым амортизация уже была начислена в течение отчетного периода, следующими документами:

  • Перемещение;
  • Изменение параметров при изменении счета накопленной амортизации;
  • Переоценка;
  • Модернизация;
  • Подготовка к передаче;
  • Списание ОС.

Гиперссылка формы документа Настройки отражения расходов для быстрого перехода к параметрам отражения в регламентированном учете будет окрашена в красный цвет в случае отсутствия счетов отражения расходов по объектам подлежащим амортизации. Так же, наличие ошибки блокирует проведение документа, требуется заполнить счета отражения расходов перед выполнением расчета.
