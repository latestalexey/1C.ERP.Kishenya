Возврат материалов из производства

Документ предназначен для оформления возврата материалов из производства на склад.

  • При заполнении документа необходимо уточнить Организацию, Подразделение и Склад.
  • Затем следует использую команду Заполнить - подобрать по остаткам в подразделении, указать те материалы, которые возвращаются из производства. В списке будут отображены только те материалы, которые передавались указанному в документе подразделению.
  • При помощи команды Перенести в документ табличная часть документа будет заполнена возвращаемыми материалами автоматически.
