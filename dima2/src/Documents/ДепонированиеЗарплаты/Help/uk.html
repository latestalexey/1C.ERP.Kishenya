Депонирование зарплаты

Документ Депонирование зарплаты предназначен для регистрации неполученных сотрудниками сумм заработной платы как депонентов. Депонирование вводится на основании ведомостей (Ведомость в кассу, Ведомость выплаты через раздатчика) или заявок на расходование денежных средств.

По депонированным суммам регистрируются:

  • очередные депоненты в книге депонентов;
  • уменьшается задолженность организации перед сотрудником, т.е. депонированные суммы считаются выплаченными;
  • если за сотрудником, для которого регистрируются депоненты, имеется задолженность по НДФЛ, регистрируются суммы по удержанию НДФЛ;
  • сумма к выплате по заявке на расходование денежных средств (если используются) при депонировании не уменьшается, последующая выплата депоненту будет выполнена в рамках существующей заявки.

Выплата депонентам осуществляется расходным кассовым ордером с указанием ведомости, по которой выполняется выплата.
