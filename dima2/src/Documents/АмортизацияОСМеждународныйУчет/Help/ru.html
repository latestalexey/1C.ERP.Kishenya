Амортизация ОС (международный учет)

Документ предназначен для отражения в системе операций по расчету амортизации основных средств, для которых стоимость погашается посредством начисления амортизации и по которым амортизация не была начислена в течение отчетного периода другими документами. Например, Списание ОС.
