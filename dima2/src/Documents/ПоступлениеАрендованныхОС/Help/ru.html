Поступление арендованных ОС

При создании документа необходимо заполнить следующие реквизиты:

На закладке Основные средства указывается перечень поступивших основных средств. Для каждого из которых указываются следующие реквизиты:

  • Залоговая стоимость - величина залоговой стоимости основного средства в валюте регламентированного учета;
  • Инвентарный номер - инвентарный номер, по которому объект будет принят к забалансовому учету.
