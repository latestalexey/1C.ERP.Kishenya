Экземпляры бюджетов

Документ предназначен для ввода плановых значений по выбранному Виду бюджета за определенный период в разрезе Сценария, а так же Организации и Подразделения, если они настроены в Модели бюджетирования.

Документ является основным инструментом для установки значений по Статьям бюджетов с детализацией до аналитики. Документ используется для первичного ввода плановых данных по бюджетам и для корректировок.

Дата документа, введенные планы (или изменения планов) используются в бюджетных отчетах, позволяя получать различные версии бюджетов на дату актуальности отчета.

В списке предусмотрены отборы:

  • Период - указание временного периода для выбора действующих в нем бюджетов
  • Вид бюджета - отбор бюджетов определенного вида
  • Сценарий - отбор по сценарию планирования
  • Статус - ограничение списка отображаемых бюджетов по статусу
  • Ответственный - отбор по сотруднику, который готовил бюджеты


С помощью команды Установить статус можно присвоить статусы сразу нескольким документам.

Если для вида бюджета установлены дополнительные аналитические разрезы, то при отборе по данному виду бюджетов в список экземпляров бюджета будут добавлены дополнительные поля со значениями использующихся аналитических разрезов.

Пример. Для вида бюджета установлен дополнительный аналитически разрез Контрагенты. В списке экземпляров бюджетов при отборе по данному виду бюджетов добавится поле Контрагенты. При выводе общего списка экземпляров бюджетов по другим видам - поле будет автоматически скрыто.
