Регистрация дефекта

Документ Регистрация дефекта предназачен для отражения фактов нарушения работоспособности объектов эксплуатации и их составных частей, а так же состояний, не удовлетворяющих установленным требованиям режима эксплуатации - снижающих функционирование в основных областях деятельности.

В документе, помимо номера и даты регистрации дефекта, указываются следующие поля:

  • Объект эксплуатации - объект, для которого регистрируется дефект;
  • Краткое описание - сжатое представление поломки, для быстрой навигации в форме журнала дефектов;
  • Вид дефекта - заданное значение классификации дефектов из заданных элементов справочника Виды дефектов;
  • Статус - состояние выполнения работ по описанному дефекту;
  • Устранить до - планируемая или требуемася дата устранения дефекта;
  • Способ устранения - метод устаранения дефекта, для определения необходимости выполнения заказа на ремонт.

На закладке Дефектные узлы может быть детализированно расписан дефект через перечисление составных частей объекта эксплуатации и изложение имеющихся неисправностей.
