Процесс подготовки бюджетов

Документ предназначен для запуска бюджетного процесса вручную.
Если на Модели бюджетирования установлено, что задачи бюджетного процесса формируются автоматически - тогда документ используется для пересчетов уже пройденного бюджетного процесса или для предварительного расчета бюджетов, даже если предыдущие этапы подготовки бюджетов выполнены не до конца. В случае, если на Модели бюджетирования указано, что бюджетный процесс запускается вручную - тогда задачи по бюджетному процессу будут формироваться только на основании документа Процесс подготовки бюджетов.

  • Модель - модель бюджетирования, бюджетный процесс которой требуется запустить
  • Группа этапов - группа этапов, по шагам которой будет запущено выполнение задач
  • Период - период, за который будут созданы бюджеты при выполнении задач. В процесс подготовки бюджетов можно включить только этапы, периодичность которых меньше или равна выбранной периодичности

Если процесс подготовки завершен и есть необходимость пересчитать бюджеты - нужно создать ноый документ Процесс подготовки бюджетов. Если в информационной базе уже имеются бюджеты за выбранный период - то при выполнении бюджетных задач пользователю будет предложено перезаполнить уже имеющиеся бюджеты, а не создавать новые.
