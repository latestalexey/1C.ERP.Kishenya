Перемещение материалов и работ в производстве

Документ отражает перемещение материалов и работ между производственными подразделениями.
Документы оформляются в рабочем месте Получение и возврат материалов.

Добавление материалов (работ) в документ
  • Для добавления нового материала (работы) нажмите на кнопку Добавить в командной панели табличной части документа.
  • Новая номенклатурная позиция добавляется путем выбора из справочника Номенклатура.
  • Если для материала, который перемещается между подразделениями, используются характеристики/упаковки, то укажите характеристику/упаковку для каждого материала.
  • Для каждого материала (работы) в поле Количество укажите количество перемещаемого материала (работы).

Регистрация документа

Регистрация документа производится при выполнении команды Провести. Перед проведением документа контролируется заполнение всех обязательных реквизитов: организация, отправитель, получатель, ответственный и т.д.

Документ Перемещение материалов и работ в производстве может проводиться в следующих статусах.

  • К отгрузке - материалы подготовлены к отгрузке из подразделения-отправителя.
  • Отгружено - товар отгружен из подразделения-отправителя.
  • Принято - товар принят в подразделение-получатель.


Использование статусов позволяет более детально отслеживать процесс перемещения материалов (работ).

Работа с терминалом сбора данных


Для того чтобы загрузить данные из терминала сбора данных необходимо воспользоваться командой табличной части Все действия/Загрузить данные из ТСД. Если терминал сбора данных подключен и настроены параметры его подключения, то табличная часть заполниться данными из терминала. В противном случае программа выдаст предупреждение. Если из терминала будет получен штрих код не зарегистрированный в информационной базе, то программа предложит зарегистрировать его и сопоставить с имеющимся в информационной базе материалом (работой).

Команда Загрузить товары из ТСД доступна в документе только при условии его оформления без заказа (не на основании заказа).

Добавление данных с помощью сканера


Если сканер подключен и настроены параметры его подключения, то после считывания штрих кода в табличную часть будет добавлен материал (работа) соответствующий штрих коду. Если материал (работа) уже присутствует в табличной части, то будет увеличено его количество. Если в программе не зарегистрирован считываемый штрих код, то программа предложит зарегистрировать его и сопоставить имеющемуся в информационной базе материалу (работе).

Для того что бы выполнить поиск материала (работы) в табличной части, необходимо воспользоваться командой табличной части Все действия/Поиск по штрихкоду. Далее в открывшейся форме считать штрих код. Если материал (работа) со считанным штрих кодом присутствует в табличной части, то он будет выделен (на него будет установлен курсор). Если в программе не зарегистрирован считываемый штрих код, то программа предложит зарегистрировать его и сопоставить имеющемуся в информационной базе материалу (работе).
