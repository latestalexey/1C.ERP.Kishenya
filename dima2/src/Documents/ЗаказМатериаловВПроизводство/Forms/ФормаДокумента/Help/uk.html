Заказ материалов в производство

Документ предназначен для регистрации потребностей со стороны подразделений предприятия в материалах. Исполнением заказа является документ Передача материалов производство.

На основании заказа могут быть созданы заказы на передачу материалов в производство, если заказанный товар производится предприятием.

Статус К обеспечению дает возможность провести обеспечение (резервирование) материалов.
Статус К выполнению дает возможность начать отгрузку заказа для тех строк, у которых установлен вариант обеспечения (колонка Действия) Отгрузить или Отгрузить обособленно.
Если необходимо контролировать завершение работы по документу, возможно использование статуса Закрыт. Проведение документа в этом статусе доступно лишь в том случае, если количество запланированных к отгрузке товаров, совпадает с количеством отгруженных товаров (по документу Передача материалов в производство, а так же по документу Расходный ордер на товары (если для склада используется ордерная схема)).

Документ может быть создан в ручном (команда Создать-Новый) и автоматическом режимах (команда Создать - По плану).

Управление статусами заказа
  • К обеспечению - если по заказу существует потребность и ее необходимо обеспечить к определенной дате.
  • К выполнению - если потребность по заказу уже обеспечена (материал есть в наличии на складе) и его можно отгрузить к тем датам, которые указаны в табличной части документа.
  • Закрыт - если заказ отгружен.

Заполнение табличной части


Поле Назначение табличной части Материалы определяет назначение обособленного материала, списываемого со склада и передаваемого в производственное подразделение.

Возможны следующие варианты передачи обособленно материала:

  • Передача необособленного материала. При такой передаче поля Действие и Назначение в заказе и передаче не заполняются.
  • Обособление материала из свободного остатка на складе и обособление в производстве. Поле Действие в заказе устанавливается в не обособленно, а Назначение заполняется назначением, под которое материалы передаются в производство.
  • Обособление под заказ на передачу и снятие обособления при передаче. Поле Действие в заказе устанавливается в Обособленно, Назначение не заполняется.
  • Передача обособленного материала. Поле Действие в заказе устанавливается в Обособленно, Назначение заполняется назначением обособленного материала на складе. Под это назначение материал будет обособлен в производстве.


Новая номенклатурная позиция добавляется путем выбора из справочника Номенклатура (кнопка Добавить). Для добавления нескольких позиций в табличную часть документа можно также использовать диалоговое окно подбора (кнопка Заполнить - Подобрать товары).

  • Для материалаов передаваемых в производство можно указать группу (вид) продукции для производства которой данные материалы передаются.


Контроль и корректировку графика отгрузки заказов можно осуществлять при помощи команды Состояние обеспечения. После выполнения рекомендаций результат переносится в редактируемый заказ по команде Перенести в документ.

С помощью команды Заполнить обеспечение определяется стратегия по обеспечению и отгрузке товаров по заказу. Доступны следующие варианты:

  • Отгрузить.
  • Резервировать на складе.
  • Резервировать по мере поступления.
  • Обеспечивать обособлено.
  • К обеспечению.
  • Не обеспечивать.


Из группы команд Функции можно заполнить статьи расходов для всего заказа или только для выделенных строк.

Корректировка обеспечения

При необходимости произвести обособленное резервирование товара со свободных остатков склада или изъять резерв другого заказа в пользу текущего, либо снять резерв текущего заказа (в свободный остаток склада) можно воспользоваться следующими командами меню Обеспечение закладки Товары:

  • Снять резерв по назначению
  • Зарезервировать под назначение

Корректировка заказа


В любой момент времени можно откорректировать количество в заказе и перепровести документ. При этом будет автоматически откорректированы данные по обеспечению или выполнению заказа. Для любой позиции в заказе можно установить признак Отменено. После проведения документа эта информация отобразится в графике движения товаров, менеджер по логистике будет иметь оперативную информацию об изменении заказа.

С помощью команды Отменить неотработанные строки можно выставить признак Отменено на те позиции, работа по которым не была произведена.

Механизм контроля
В том случае, если не совпадает запланированное и отгруженное количество по товарным позициям, механизм контроля запрещает проведение.
Запланированное количество рассчитывается по товарным позициям заказа. Отгруженное количество по товарным позициям накладной. В том случае, если используется ордерная схема, дополнительно проверяется совпадение запланированного количества с количеством по ордеру.
