Изделия и затраты НЗП

Документ предназначен для ввода информации о материалах, которые фактически потрачены на производство продукции, выпуск которой еще не отражен в системе.
Данная информация регистрируется в системе только для продукции, выпуск которой оформляется без распоряжений, поскольку при производстве по заказу на производство информация о потребленных материалах фиксируется в маршрутных листах.
Документ вводится в конце отчетного периода при длительном цикле производства. Информация о потребленных материалах используется в рабочем месте Распределение материалов и работ для заполнения колонки Лимит в обработке. Лимит в обработке показывает, какое количество материалов является допустимым на конец отчетного периода.
В шапке документа указываются реквизиты продукции, а в табличных частях материалы, израсходованные на производство, и трудозатраты. Табличные части могут быть заполнены по спецификации.

Поле Количество определяет количество изделий в незавершенном производстве. На этот объем выполняется расчет материалов и трудозатрат при заполнении табличных частей по спецификации.
