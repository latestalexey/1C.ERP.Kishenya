Передачи давальцам

Форма предназначена для оформления передач продукции давальцам, контроля расхождений с расходными ордерами на товары. Форма состоит из двух закладок:

  • Оформлено - список оформленных накладных. Непосредственный ввод передач недоступен, для создания новых документов следует выбрать распоряжения на передачу. В списке предусмотрен отбор по менеджеру.
  • К оформлению - распоряжения на оформление передач давальцам. Это список Заказов давальцев в разрезе складов, по которым возможна отгрузка.

В списке распоряжений предусмотрены индикаторы:

  • Состояние оформления накладной показывает, на всю ли продукцию по заказу давальца оформлена накладная. Индикатор имеет три состояния: накладные еще не оформлены, накладные оформлены, но не на все количество по заказу и накладные оформлены полностью.
  • Соответствие расходных ордеров накладным показывает, есть ли расхождения между оформленными накладными по заказу и расходными ордерами на товары. Индикатор имеет три состояния: ордера не оформлены, есть расхождения и расходные ордера соответствуют накладным.

Оформление по распоряжению

Оформление по распоряжению

Для оформления новой передачи давальцу (накладной) требуется перейти на закладку К оформлению.

Документ может быть создан на основании одного или нескольких Заказов давальца. Оформление передач по нескольким заказам регулируется функциональной опцией Отчет и передача давальцу по нескольким заказам в разделе НСИ и администрирование - Продажи - Производство из давальческого сырья. Оформление передач становится доступным по мере установки действия по обеспечению Отгрузить обособленно в строках продукции заказа давальца, сам заказ при этом должен быть в статусе К отгрузке. Количество переданной продукции не может превышать количество заказанной.

  • Команда Оформить по заказу предназначена для создания новой накладной на продукцию, готовую к оформлению по заказу давальца.
  • Команда Оформить по отгрузке предназначена для оформления накладной по складским ордерам. При этом, если по заказу уже введена хотя бы одна накладная, то открывается вспомогательная форма Переоформление передач давальцам. С помощью этой формы можно оформить новую накладную или перезаполнить существующую по приемке.

По состояниям накладных и соответствию ордеров доступны отборы.
